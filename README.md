# magaluE2E
Testes E2E - cucumber, capybara e ruby

## Como executar o projeto

* Importante ter o ruby instalado (versão 2.5 ou superior)

### Instalar o bundler
`
gem install bundler
`

### Instalar as dependências do Ruby (projeto)
`
bundle install
`

### Para executar localmente 
`
bundle exec cucumber
